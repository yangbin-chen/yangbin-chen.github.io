# Yangbin Chen — Academic Homepage

Personal academic homepage migrated from [yangbin-chen.github.io](https://yangbin-chen.github.io/).

## Quick start

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Deploy to GitHub Pages

Push this folder to the `Yangbin-Chen.github.io` repository (or any static host). If replacing the old site, also copy the `pdf/` folder from the previous repo for slide links to work locally.
